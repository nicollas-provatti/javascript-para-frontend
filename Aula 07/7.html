DOM

O DOM é a sigla para Document Object Model (Modelo de Objeto de Documento), uma interface de programação que representa um documento HTML, XML ou SVG como uma estrutura de árvore lógica, onde cada parte do documento é um nó. Acesse MDN Web Docs para saber mais.

Tópicos importantes

  • Seleção de elementos
  • Manipulação de elementos
  • Eventos
