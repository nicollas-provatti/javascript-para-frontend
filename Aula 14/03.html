🔵 Manipulação do DOM

No DOM, podemos fazer manipulações mais complexas como acessar parentes específicos de um elemento (pai, irmão, etc) e criar tags HTML e adicionar usando JavaScript.

🔍 Como criar e adicionar elementos ?

É possível criar novos elementos dinamicamente e adicioná-los à página. Isso é útil quando queremos modificar ou expandir a estrutura da nossa página sem recarregar a página inteira. Aqui estão os principais métodos para criar e adicionar elementos ao DOM:

  • O método createElement() é usado para criar um novo elemento HTML.
  • O método appendChild() permite que você adicionar um nó como um filho de outro elemento.
  • O método cloneNode() cria um cópia exata do nó. A cópia pode ser profunda ou superficial.

🔍 Como remover e substituir elementos ?

Além de criar e adicionar elementos ao DOM, é igualmente importante saber remover e substituir elementos existentes. Isso permite que o conteúdo da página seja dinâmico e interativo. Aqui estão os principais métodos:

  • O método removeChild() remove um elemento filho de um elemento pai.
  • O método remove é mais direito. Ele remove o próprio elemento sem precisar do pai.
  • O método replaceChild() permite substituir um filho existente por outro.
